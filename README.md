# Arminder Singh -- Portfolio

My personal developer portfolio, showcasing projects, skills, and experience.

**Live site: [arminderf-singh.github.io/MyPortfolio](https://arminderf-singh.github.io/MyPortfolio)**

---

## About

A responsive personal portfolio built to showcase my software engineering work including projects, tech stack, and experience. Designed with recruiters and collaborators in mind.

---

## Features

- Project showcase with live demos and GitHub links
- Fully responsive across desktop, tablet, and mobile
- Fast load times with a Vite-powered build and optimized assets
- Direct GitHub repository links for all featured projects
- Clean, developer-focused design

---

## Tech Stack

| Layer | Technology |
|---|---|
| Frontend | React, TypeScript |
| Styling | Tailwind CSS |
| Build Tool | Vite |
| Hosting | GitHub Pages, Cloudflare |

---

## Getting Started

```bash
git clone https://github.com/Arminderf-Singh/MyPortfolio.git
cd MyPortfolio
npm install
npm run dev
```

To build for production:

```bash
npm run build
```

---

## Deployment

This site is deployed via GitHub Pages. Push the `dist/` folder to your `gh-pages` branch, or configure GitHub Actions for automatic deployment on push.
